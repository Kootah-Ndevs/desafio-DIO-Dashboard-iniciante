# Dashboard de Vendas - Projeto Bootcamp

## 📋 Descrição do Projeto
Desenvolvimento de um dashboard de vendas (Xbox Game Pass / EA Play / Minecraft) utilizando Microsoft Excel para visualização de métricas de subscrição e performance.

## 🛠️ Tecnologias Utilizadas
* Microsoft Excel (Versão Web / Office 365)
* Técnicas de ETL básico (Cálculos e Formatação de Moeda)
* Gráficos de Barras e Cards de "Big Numbers"

## ⚠️ Observações Técnicas (Limitações)
Este projeto foi desenvolvido integralmente na versão *Excel Online*, o que impôs algumas restrições técnicas em comparação à versão Desktop:

1. *Segmentação de Dados (Slicers):* Devido a limitações de permissão e compatibilidade do Excel Online, a inserção de Slicers para filtros dinâmicos apresentou instabilidade, sendo mantida a estrutura de dados baseada em Tabelas Dinâmicas.
2. *Vínculos de Caixas de Texto (Big Numbers):* O recurso de vincular uma Caixa de Texto a uma célula para atualização dinâmica de valores é limitado na versão Web. Tentei manter a fidelidade visual às instruções, mas priorizei a integridade dos cálculos na aba de origem.
3. *Design:* O layout foi construído seguindo o guia visual do instrutor, utilizando formas e ícones para facilitar a leitura executiva.
